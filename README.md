### Tally Data Sync Backend

**Routes Usage**

* **Health Check**
* GET: `https://tally-data-backend.onrender.com/health`


* **Tally Raw (Raw JSON Dumps)**
* GET: 
`https://tally-data-backend.onrender.com/api/tally/raw?collectionName=YOUR_COLLECTION_NAME`
* POST: 
`https://tally-data-backend.onrender.com/api/tally/raw?collectionName=YOUR_COLLECTION_NAME`
